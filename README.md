# crawl-mm
* 简单百度美女图片爬虫

## 快速开始
### 系统依赖

```shell
# RHEL/CentOS
# yum install curl wget awk jq
```

### 开始安装

* 默认图片下载位置为: ~/image.baidu.com/images

* curl https://raw.githubusercontent.com/jsbd/crawl-mm/master/crawl.sh | bash
